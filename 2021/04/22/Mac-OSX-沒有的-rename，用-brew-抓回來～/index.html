Mac OSX 沒有的 rename，用 brew 抓回來～

发表于 2021-04-22 分类于 编程 阅读次数： Valine：
本文字数： 1.3k 阅读时长 ≈ 1 分钟

brew 內剛好有 rename 這個套件，但使用方法大不同

安裝 OSX rename

$ brew install rename

但是此 rename 非彼 rename，用法和 Linux 上的並不相同
